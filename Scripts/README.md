# სალამურას თავგადასავალი
## პროექტის შესახებ
**სალამურას თავგადასავალი** არის Unity-ში შექმნილი 3D საგანმანათლებლო ვიდეო თამაში, რომელიც დაფუძნებულია ქართულ ანიმაციურ მულტფილმზე „სალამურას თავგადასავალი“.
პროექტის მთავარი მიზანია სწავლისა და გართობის გაერთიანება. თამაშში ინტეგრირებულია კითხვა-პასუხის სისტემა, რომელიც იყენებს განათლების ფსიქოლოგიის ცნობილ პრინციპს - „active recall“-ს, რომლის თანახმადაც ინფორმაციის არა მხოლოდ გამეორება, არამედ გახსენებაც განამტკიცებს გრძელვადიან მეხსიერებას.
თამაში შექმნილია Unity Engine-ისა და C# პროგრამირების ენის გამოყენებით, ხოლო 3D მოდელები შექმნილია Blender-ში.

---
## ძირითადი ფუნქციონალი

### საგანმანათლებლო კითხვა-პასუხის სისტემა

* მომხმარებელს შეუძლია დაამატოს საკუთარი კითხვები და პასუხები.
* კითხვები ავტომატურად ნაწილდება დონეებზე.
* თამაშის მიმდინარეობისას ჩნდება შემთხვევითად შერჩეული კითხვები.
* სწორი პასუხის გაცემამდე თამაში ჩერდება.
* პასუხების მდგომარეობა ინახება მიმდინარე სესიის ფარგლებში.
* კითხვები ნაწილდება დონეებს შორის შესაბამისი ალგორითმით.

## თამაშის დონეები

### Level 1 

ტყის გარემოში მოთამაშემ უნდა:

* შეაგროვოს ჭიამაიები;
* თავი აარიდოს შხამიან სოკოებს;
* არ ჩამოვარდეს კოშკიდან;
* შეაგროვოს ყველა ჭიამაია შემდეგ დონეზე გადასასვლელად.

ფუნქციები:

* Bug Collection System
* Health System
* Ladder Climbing
* Terrain Physics
* 3D Audio Effects

---

### Level 2 

მოთამაშემ უნდა:

* შეაგროვოს მუსიკალური ნოტები;
* გადალახოს დაბრკოლებები;
* მიაღწიოს სასახლეს.

დაბრკოლებები:

* ეკლიანი ლიანა (დახრა)
* წაქცეული ხე (ახტომა)
* დიდი ხე (გვერდის აცილება)

ფუნქციები:

* Procedural Road Generation
* Endless Runner Mechanics
* Note Collection System
* Sound Effects
* Restart System

---

### Level 3 

მოთამაშემ უნდა:

* შეასრულოს მალაყები და დაუკრას სალამურზე;
* მოასწროს დავალებების შესრულება დროის ამოწურვამდე.

ფუნქციები:

* Timed Gameplay
* Flip Mechanics
* Animation Events
* Restart System

---

### Level 4 

მოთამაშის მიზანია:

* გადაადგილდეს მოძრავ პლატფორმებზე;
* თავი აარიდოს დაბრკოლებებს;
* დროის მოსამატებლად აიღოს ელექსირი;
* დროის ამოწურვამდე ჩავიდეს საბოლოო პლატფორმაზე.

ფუნქციები:

* Tile-Based Platform System
* Obstacle Interaction
* Countdown System
* Save/Restore Time Logic
